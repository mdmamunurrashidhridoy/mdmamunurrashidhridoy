I'm a **Computer Science and Engineering** graduate with a sharp focus on building reliable, well-architected software systems. My work lives at the intersection of **backend engineering**, **artificial intelligence**, and **systems thinking** — where clean code meets real-world complexity.

I approach every project with a bias toward clarity: clean abstractions, intentional design, and code that's built to last. Whether it's a production API, an ML pipeline, or a Django backend, I'm interested in software that scales — technically and conceptually.
